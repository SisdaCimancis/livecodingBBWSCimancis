Silahkan Buat aplikasi Todo List dengan fitur:

1. Menampilkan daftar todo

2. Menambah todo baru

3. Menandai todo sebagai selesai

4. Menghapus todo

Gunakan HTML, CSS, dan JavaScript untuk membuat aplikasi sederhana ini.
Pastikan aplikasi memiliki antarmuka yang user-friendly dan fungsionalitas yang sesuai dengan fitur yang diminta.

Penilaian akan didasarkan pada:

1. Kelengkapan fitur sesuai dengan yang diminta

2. Kualitas kode (struktur, kebersihan, dan penggunaan best practices)

3. Desain antarmuka pengguna

Front-End : HTML + CSS + JavaScript (Framework boleh, tapi tidak wajib)

Backend: Bebas (Node.js / PHP / Python) atau simulasi data di frontend (untuk FG)

Selamat Mengerjakan!
